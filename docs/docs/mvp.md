# MVP

## Goal

Build a Telegram bot that helps users quickly decide what they want to eat.

The MVP is successful if a user can receive a relevant dish recommendation in less than one minute.

---

## Target User

A person who:

- Doesn't know what they want to eat.
- Doesn't want to browse dozens of restaurant menus.
- Wants a quick and confident recommendation.

---

## Features Included

### Mode 1 — Quick Choice

The user selects several tags, for example:

- Spicy
- Meat
- Hot
- For sharing

The bot recommends one dish.

---

### Mode 2 — I Have No Idea

The bot asks a short series of questions.

Each answer narrows down the recommendation.

The goal is to identify the user's current craving in as few questions as possible.

---

## Recommendation Screen

The bot shows:

- One dish
- A short explanation of why it was recommended
- Buttons:
  - ✅ That's it!
  - 🔄 Another option

---

## Feedback

After the recommendation, the user can indicate whether they liked it.

This feedback will improve future recommendations.

---

## Not Included

The first version does NOT include:

- Restaurant search
- Delivery services
- Maps
- Prices
- Reservations
- User accounts
- Personal recommendation history

These features will be added only after validating the core idea.
